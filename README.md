# Geldautomat
Spender für Taschengeld

Hier entsteht ein Auszahkungsautomat für das Taschengeld der Kinder. Es werden ausschließlich 50ct-Münzen ausgezahlt. Das System besteht aus einer Servo-betriebenen Mechanik, einem Arduino-Interface für den Servo mit serieller Ansteuerung aus einem Raspberry Pi.
